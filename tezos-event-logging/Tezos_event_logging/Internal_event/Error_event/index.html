Module Internal_event.Error_event

Error_event.t is a generic event to emit values of type Error_monad.error list.

type t = {
message : string option;
severity : [ `Fatal | `Recoverable ];
trace : Tezos_error_monad.Error_monad.error list;
}

Errors mainly store Error_monad.error list values. One can attach a message and a severity (the default is `Recoverable which corresponds to the Error level, while `Fatal corresponds to Fatal).

val log_error_and_recover : ?⁠section:Section.t -> ?⁠message:string -> ?⁠severity:[ `Fatal | `Recoverable ] -> (unit -> (unit, Tezos_error_monad.Error_monad.error list) Stdlib.result Lwt.t) -> unit Lwt.t

log_error_and_recover f calls f () and emits an Error_event.t event if it results in an error. It then continues in the _ Lwt.t monad (e.g. there is no call to Lwt.fail).
